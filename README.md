### 1. K კოეფიციენტის გამოთვლა
* **სტუდენტის სახელი და გვარი:** გიორგი ჩეკურიშვილი
* **დაბადების რიცხვი:** 12
* **სახელის სიგრძე:** 6 (გიორგი)
* **გვარის სიგრძე:** 11 (ჩეკურიშვილი)

**K კოეფიციენტის ფორმულა:**
* K = (სახელის ასოების რაოდენობა + გვარის ასოების რაოდენობა) / დაბადების რიცხვი
* ჩემი K კოეფიციენტი: (6 + 11) / 12 ≈ 1.416

**საბოლოო დანაზოგის (Final Savings) ფორმულა:**
* დანაზოგი = (შემოსავალი - ხარჯები) * 1.416
